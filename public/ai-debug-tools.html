Instructions

  1. Click "Go to Dashboard" to navigate to the main dashboard
  2. Once on the dashboard, click "Inject Debug Script" to activate the debug tools
  3. Use the debug controls that appear in the bottom right to:
    • "Find AI Components" - Highlight all AI-related elements currently visible
    • "Force Visible" - Make hidden AI components visible
  4. Navigate to different sections of the dashboard to find all AI features
  5. Click the highlighted borders to remove them
Note: If AI components are not found, try navigating to different tabs or sections. They might be conditionally rendered based on the current view.

Manual Debug Script

If the button doesn't work, copy and paste this code into your browser console:

const script = document.createElement('script'); script.src = '/ai-debug-direct.js'; document.body.appendChild(script);

Check Browser Console

Open your browser's developer tools (F12 or Right-click > Inspect) and check the console for debug messages and component information.
