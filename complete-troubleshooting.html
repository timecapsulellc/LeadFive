üîß Common Issues & Solutions

1. "Port already in use"

# Kill processes on port 5175 lsof -ti:5175 | xargs kill -9 # Or use alternative port npm run dev:3000

2. "Permission denied"

# Fix permissions sudo chown -R $(whoami) "/Users/dadou/LEAD FIVE" chmod +x node_modules/.bin/vite

3. "Module not found"

# Reinstall dependencies rm -rf node_modules package-lock.json npm install

4. "Command not found: npm"

Install Node.js from nodejs.org

5. Still blank page after server starts

# Hard refresh browser # Mac: Cmd + Shift + R # Windows: Ctrl + Shift + F5 # Or open incognito/private window
